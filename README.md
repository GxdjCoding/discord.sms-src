# Discord Sms ( BETA )

โมดูลสำหรับสร้างบอทยิงเบอร์ และบอทอื่นๆ ใน Discord เช่น บอทกันยิงดิส !!

# How to install

`npm install discord.sms`

# How to use

`node`

```js
const { connect, login, sms_command, cooldown } = require('discord.sms');

connect();

login();

sms_command();

cooldown();
```
